# 第六周：神经网络
## 内容
### 线下报告 - 强化学习
- 主要内容
    - 周三：对强化学习（reinforcement learning）的初步介绍
        - Markov 决策过程
        - Q-learning 
        - 应用：吃豆人游戏等
    - 周五：深度强化学习介绍
        - Policy Gradient & REINFORCE 算法
        - A3C 算法
        - 模仿学习（Imitation Learning）
        - 应用：Atari 游戏, AlphaGo 围棋等

- 开始时间：本周三、周五（8月23、25日）上午 9:40
- 地点：东校区会议室
